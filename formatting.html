Workshop: HTML Text Formatting


ฝึกใช้เเท็กรูปแบบข้อความ (Formatting Tags) เพื่อสร้างเนื้อหาที่อ่านง่ายเเละเป็นระเบียบ


1. สิ่งที่ต้องรู้ก่อนเริ่มทำ workshop

HTML มีแท็กสำหรับจัดรูปแบบข้อความ ทำให้เนื้อหาดูเด่น อ่านง่าย และสื่อความหมายได้ดีขึ้น

  • • <b>- ตัวหนา (เน้นเฉยๆ)
  • • <strong>- ตัวหนาเเบบ "สำคัญ" เชิงความหมาย (semantic importance)
  • • <i>- ตัวเอียง (ใช้กับศัพท์เฉพาะหรือตัวเน้น)
  • • <em>- ตัวเอียงเเบบเน้นน้ำเสียง
  • • <u> - ขีดเส้นใต้
  • • <ins> - ขีดเส้นใต้
  • • <mark> - ไฮไลด์ข้อความ
  • • <code> - แสดงข้อความเป็นรูปแบบโค้ด
  • • <small> - ข้อความขนาดเล็ก
  • • <del> - ข้อความขีดฆ่า
  • • <s> - ข้อความขีดฆ่า
  • • <sub> - ตัวห้อย เช่น H2O
  • • <sup> - ตัวยก เช่น x2

แท็กที่รองรับใน HTML5


  • • <strike>- not support in HTML5
  • • <big>- not support in HTML5
  • • <tt>- not support in HTML

เทคโนโลยีคอมพิวเตอร์สมัยใหม่

  ปัจจุบัน ปัญญาประดิษฐ์ (AI) มีบทบาทสำคัญในการช่วยวิเคราะห์ข้อมูลและสนับสนุนการตัดสินใจ ทำให้การทำงานมี ประสิทธิภาพสูงขึ้น และลดเวลาในการดำเนินงานในหลายอุตสาหกรรม

  เทคโนโลยี Cloud Computing ช่วยให้ผู้ใช้งานสามารถจัดเก็บและเข้าถึงข้อมูลผ่านอินเทอร์เน็ตได้ทุกที่ทุกเวลา พร้อมทั้งรองรับการทำงานร่วมกันได้อย่าง สะดวกและรวดเร็ว

  นอกจากนี้ Cybersecurity ยังมีความสำคัญในการป้องกันข้อมูลจากการโจมตีทางไซเบอร์ โดยผู้ใช้งานควรตั้งรหัสผ่านที่แข็งแรง เช่น P@ssw0rd123 และอัปเดตระบบอยู่เสมอเพื่อเพิ่มความปลอดภัย

ซอฟต์แวร์ (Software)

ซอฟต์แวร์คือ ชุดคำสั่ง ที่ใช้ควบคุมการทำงานของคอมพิวเตอร์ช่วยให้ผู้ใช้สามารถทำงานได้อย่างสะดวก และรองรับการใช้งานในหลายด้าน เช่น การศึกษา การทำงาน และความบันเทิง

ซอฟต์แวร์แบ่งออกเป็น ซอฟต์แวร์ระบบ และ ซอฟต์แวร์ประยุกต์ โดยระบบปฏิบัติการทำหน้าที่ควบคุมเครื่อง ส่วนโปรแกรมประยุกต์ใช้สำหรับงานเฉพาะ เช่น พิมพ์เอกสารหรือท่องอินเทอร์เน็ต

การเลือกใช้ซอฟต์แวร์ควรคำนึงถึง ความเหมาะสม,ความปลอดภัย และ การอัปเดตอย่างสม่ำเสมอ เพื่อให้การใช้งานมีประสิทธิภาพและลดความเสี่ยงจากปัญหาด้านความปลอดภัย
